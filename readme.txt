Re-send welcome e-mails to users who did not receive the original welcome e-mail on registration.


== Installation ==

1. Upload `resend-welcome-email.php` to the `/wp-content/plugins/wp-resend-welcome-email` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
